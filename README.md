# ofxHistogramMatching
Histogram Matching for Openframeworks using ofxCv

![HMat example]( https://github.com/bemoregt/ofxHistogramMatching/blob/master/result.png "HMat1") 

### Dependency
- OpenFrameworks 0.10.1
- ofxCv
- ofxOpenCv
- ofxGUI
- XCode 10.12.x
